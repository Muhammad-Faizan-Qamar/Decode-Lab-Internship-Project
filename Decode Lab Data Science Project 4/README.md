# 🎬 NLP & Sentiment Analysis

## 📌 Project Overview

This project is part of the **DecodeLabs Data Science Internship (Project 4)**. The objective is to build a **Natural Language Processing (NLP)** pipeline that analyzes movie reviews and predicts whether a review expresses **Positive** or **Negative** sentiment.

The project demonstrates the complete workflow of text classification, including text preprocessing, TF-IDF vectorization, machine learning model training, evaluation, and sentiment prediction.

---

## 🎯 Objectives

* Load and explore a real-world text dataset.
* Clean and preprocess unstructured text data.
* Convert text into numerical features using TF-IDF.
* Train multiple machine learning models for sentiment classification.
* Evaluate model performance using standard classification metrics.
* Predict sentiment for new user-provided reviews.
* Save the trained model for future use.

---

## 📂 Dataset

* **Dataset:** IMDb Movie Reviews Dataset
* **Total Records:** 50,000 Reviews
* **Features:**

  * **review** – Movie review text
  * **sentiment** – Positive / Negative
