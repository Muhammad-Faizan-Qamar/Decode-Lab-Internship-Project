# 📊 Advanced EDA & Feature Engineering

This project is part of the **DecodeLabs Industrial Training Kit (Batch 2026)**. The objective is to transform a raw dataset into a clean, high-quality dataset ready for Machine Learning by performing **Exploratory Data Analysis (EDA)** and **Feature Engineering**.

## 🎯 Project Objectives

* Perform Exploratory Data Analysis (EDA)
* Handle missing values using appropriate techniques
* Detect and treat outliers using the IQR method
* Create new meaningful features from existing data
* Prepare a clean dataset for Machine Learning models

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab

## 📌 Key Tasks Performed

* Imported and explored the dataset
* Analyzed dataset structure and statistical summary
* Identified and handled missing values
* Detected and treated outliers using the Interquartile Range (IQR) method
* Converted the Date column into datetime format
* Created new features:

  * OrderMonth
  * OrderYear
  * AverageItemPrice
  * CouponUsed
  * OrderValue
* Saved the cleaned dataset for further Machine Learning tasks
