# 🚨 Fraud Detection Pipeline

This project is part of the **DecodeLabs Industrial Training Kit (Batch 2026)** and focuses on building a **Fraud Detection Pipeline** using supervised machine learning techniques. The objective is to identify fraudulent credit card transactions from a highly imbalanced dataset by applying data preprocessing, SMOTE oversampling, and classification models.

## 📌 Project Objectives

* Load and explore the credit card fraud dataset.
* Perform data cleaning and preprocessing.
* Analyze class imbalance and apply **SMOTE**.
* Train **Logistic Regression** and **Random Forest** models.
* Tune model performance using **GridSearchCV**.
* Evaluate models using **Precision**, **Recall**, **F1-Score**, **ROC-AUC**, and **Confusion Matrix**.
* Compare model performance and select the best classifier.

## 📊 Dataset

The project uses the **Credit Card Fraud Detection** dataset from Kaggle, containing highly imbalanced transaction records with the **Class** column as the target variable:

* **0** = Normal Transaction
* **1** = Fraudulent Transaction
